<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

首先安装环境依赖库，对于FastAPI,推荐安装所有的可选依赖及对应功能
* BeautifulSoup
  ```sh
  pip install beautifulsoup4
  ```
* FastAPI
  ```sh
  pip install "fastapi[all]"
  ```

### Installation


Clone the repo
   ```sh
   git clone https://github.com/CUTEPKQ/Web-Spider.git
   ```
